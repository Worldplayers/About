PureEntitiesX

PureEntitiesX is a plugin that adds Mob AI to Pocketmine. It is intended to be a stand alone Mob AI for PMMP, but once Mob AI is officially implemented, we plan to migrate our Config and Customization options into a plugin using Pocketmine's AI.
